### 流程总览
1. 环境搭建：指导用户安装 Node.js → npm install → npm run dev
2. 收集完整问题清单：PDF已有20+问题 + 用户补充剩余问题（截图+文字）
3. 创建修复计划：按模块分组，通病优先

5. 全部验收通过后打包部署到服务器

### 单次修复验收流程
```
我修改代码 → 告知用户刷新浏览器 → 用户查看效果
  → OK → 记录通过，进入下一个

```
