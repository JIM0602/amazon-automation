关键设计决策:
- postgres 端口 5432 未映射宿主机（仅内部网络），防止数据库直接暴露
- app 以 appuser 非root身份运行，提升安全性
- 每日备份保留7天，自动删除过期文件
- 监控阈值：磁盘>80%、内存>80% 触发飞书告警
- nginx 同时支持普通 HTTP 代理和 WebSocket 升级（飞书回调需要）
- systemd 服务在 Docker 服务启动后才运行，保证依赖顺序
