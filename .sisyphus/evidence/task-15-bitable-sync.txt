## 实现要点
1. 使用 httpx.Client 作为 HTTP 客户端（支持 context manager 协议）
2. token 鉴权通过 get_bot().get_tenant_access_token() 获取（全局单例，有缓存）
3. 写操作（create/update/delete/batch_create）均记录 audit_logs：
   - action: bitable.{method_name}
   - actor: bitable_sync
   - pre_state/post_state: JSON 记录操作上下文
4. 所有方法均有 try/except，失败返回空 dict / 空 list / False，并 logger.exception 记录
5. batch_create_records 按 BATCH_SIZE=50 自动切片，每批独立发送，失败批次跳过不影响其他批次
6. list_records 支持翻页（has_more + page_token），自动循环直到数据全部获取
