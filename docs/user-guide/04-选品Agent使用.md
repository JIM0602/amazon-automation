# 04-选品分析功能：AI 助手的核心洞察

选品分析是 PUDIWIND AI 助手最强大的功能之一。它能自动收集市场数据、分析竞品趋势，并结合我们的运营逻辑，给出具体的**选品机会建议**。

---

## 🕒 什么时候用？

- **定期例行检查**：每周或每月一次，查看宠物玩具等核心品类的市场动态。
- **发现机会时**：在社交媒体看到某个宠物单品爆火，想要快速分析其在亚马逊的市场前景。
- **想要开发新品时**：在已有产品线基础上寻找关联产品的扩充建议。

## 🚀 如何触发分析？

发送指令 `选品分析 <类目或产品>` 即可。

- **单聊中**：`选品分析 宠物垫`
- **群聊中**：`@PUDIWIND运营助手 选品分析 牵引绳`

## ⏳ 系统会分析什么？

当您触发分析后，AI 助手会执行以下步骤（大约耗时 **3-5 分钟**）：
1. **收集市场现状**：通过第三方工具（如卖家精灵）获取该品类的市场规模、平均售价及增长率。
2. **抓取竞品数据**：分析该品类中排名前 20 的产品（Best Sellers）及其 BSR 排名变化。
3. **结合运营逻辑**：从资料库（Knowledge Base）中提取选品原则（如利润空间、差评点分析等）。
4. **生成最终建议**：总结出 3-5 个具体的选品方向。

## 📊 结果在哪里看？

1. **飞书消息简报**：分析完成后，AI 助手会直接在对话框中给出一份文字摘要，包含核心指标和初步选品方向。
2. **多维表格（详细数据）**：系统会自动将所有分析出的具体产品数据、竞争指标及推荐理由同步到飞书多维表格中，方便您进一步筛选和比对。

> **查看技巧**：想要看详细的竞品对比数据？请点击飞书左侧的 **"多维表格"** 应用，找到 **"PUDIWIND 选品分析表"**。详见 [06-飞书多维表格](06-飞书多维表格.md)。
