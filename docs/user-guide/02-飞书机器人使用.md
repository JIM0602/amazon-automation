# 02-飞书机器人使用说明：所有指令全集

PUDIWIND AI 助手支持一系列简单的文字指令。您可以直接在对话框中发送，或者在群聊中 @机器人 并发送。

---

## 📋 核心指令列表

| 场景 | 发送格式示例 | 预期回复 |
| :--- | :--- | :--- |
| **知识库问答** | `?如何处理差评` | 提取运营资料库中的专业建议回复您 |
| **运营日报** | `今日报告` 或 `日报` | 一张包含昨日销售额、订单数等指标的日报卡片 |
| **选品分析** | `选品分析 宠物玩具` | 分析指定类目的市场趋势，给出具体的选品机会建议 |
| **紧急停机** | `暂停所有` 或 `紧急停机` | 系统立刻停止所有正在进行的运营任务（用于处理突发异常） |
| **查看帮助** | `帮助` 或 `help` | 显示最新的指令列表说明 |

---

## 1. 知识库问答（向系统提问）

- **什么时候用**：当您对亚马逊政策、运营技巧或售后处理流程有疑问时。
- **发送格式**：
  - `?你的问题`（半角问号开头）
  - `？你的问题`（全角问号开头）
  - `问：你的问题`
- **结果**：系统会查阅内部存储的运营资料库（Knowledge Base），给出经过整理的专业回答。
- **发送示例**：
  - 发给机器人：`?亚马逊A+页面怎么优化`

![问答示例](screenshots/kb_query.png)

## 2. 运营日报（查看数据大盘）

- **什么时候用**：每天晨会前或随时想要查看店铺整体经营数据。
- **支持关键词**：`日报`、`报告`、`今日报告`、`运营报告`、`数据报告`、`每日报告`。
- **结果**：显示昨日销售额、对比前日和上周的变化趋势，以及各 SKU 销量排行和库存告警。
- **发送示例**：
  - 发给机器人：`日报`

## 3. 选品分析（市场趋势分析）

- **什么时候用**：想要开发新产品、寻找市场空白或了解竞争对手动态时。
- **支持关键词**：`选品`、`选品分析`。
- **注意**：分析约需 **3-5 分钟**，请耐心等待。
- **结果**：系统会通过飞书发送简报，并将详细数据同步到多维表格（详见 [06-飞书多维表格](06-飞书多维表格.md)）。
- **发送示例**：
  - 发给机器人：`选品分析 猫爬架`

## 4. 紧急停机（系统保护开关）

- **什么时候用**：发现系统数据异常、发送错误消息或出现任何无法预料的问题时。
- **支持关键词**：`暂停所有`、`紧急停机`。
- **效果**：系统会立即中断当前运行的所有自动化任务，并向管理层发送通知。
- **说明**：系统停机后，所有自动化流程（如每日报告发送、定时数据同步）都将停止，直到由技术人员手动恢复。

## 5. 帮助指令（命令字典）

- **什么时候用**：忘记具体怎么发指令时。
- **支持关键词**：`帮助`、`help`。

## 💬 聊天室/群聊使用技巧

1. **私聊机器人**：在与机器人的单聊窗口中，直接输入上述任何指令即可。
2. **群聊中使用**：在群里需要 **@PUDIWIND运营助手**，然后再输入指令。例如：`@PUDIWIND运营助手 选品分析 狗绳`。
