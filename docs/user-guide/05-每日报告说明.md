# 05-每日报告说明：解读运营日报

每天上午 09:00，PUDIWIND AI 助手都会自动通过飞书发送一份**运营日报卡片**。这份报告汇总了昨日的销售业绩、库存告警及市场动态。

---

## 🕒 每日报告自动发送时间

系统默认设定在 **每天上午 09:00** 发送。

- **为什么是 09:00？** 此时亚马逊美国的昨日数据已基本同步完成，方便您开始一天的工作前审视店铺。

## 🚀 如何手动触发日报？

如果您急于查看最新数据，或者由于某种原因错过了定时报告，可以直接向机器人发送指令。

- **发送格式**：
  - `日报`
  - `今日报告`
  - `运营报告`
- **结果**：系统会立刻重新生成一份最新的日报并发送给您。

## 📊 报告包含哪些核心指标？

1. **📊 销售数据汇总**：
   - **昨日总销售额**：店铺昨日全天的美金销售额。
   - **订单数 & 退款数**：昨日生成的订单数量，以及系统记录的退款件数。
   - **环比昨、上周变化**：用绿色 ↑（上升）和红色 ↓（下降）直观展示对比昨天和上周同期的百分比。
2. **📈 SKU 销量排行 (Top 3)**：
   - 展示昨日卖得最好的前 3 个产品及其销量。
3. **🤖 Agent 任务进度**：
   - 列出选品分析、数据同步等 AI 任务的运行状态（✅ 成功 / 🔄 运行中 / ⚪ 未运行）。
   - **待审批任务**：提醒您是否有需要人工确认的操作。
4. **📈 市场动态简报**：
   - **类目规模 & 增长率**：核心品类（如宠物用品）的整体市场规模和预计年增长率。
   - **库存/广告告警**：显示是否有 SKU 库存不足（如低于 20 件）或广告超支。

## 💡 如何看懂报告？

- **关注红色指标**：如果销售额环比下降超过 20%，建议点击报告下方的 **"📋 查看详情"** 深入分析原因。
- **库存告警**：一旦看到红色 🔴 告警，请立即联系采购主管安排补货。
- **选品机会**：如果您觉得日报中的类目数据表现亮眼，可以点击卡片底部的 **"🔍 触发选品分析"** 按钮，让 AI 为您深入研究该品类。
