# 亚马逊FBA库存管理策略

## 概述

库存管理是亚马逊FBA卖家运营中最容易被忽视但影响最大的环节之一。库存过多会产生高额仓储费和资金占用，库存不足（断货）则会直接导致排名下降和销售损失。科学的库存管理策略能帮助卖家在两者之间找到最优平衡点。

## FBA库存核心指标

### 1. IPI分数（Inventory Performance Index）

IPI是亚马逊对卖家库存管理效率的综合评分（0-1000分），每周更新。

**IPI评分影响因素**：
- **多余库存率**（权重最高）：库存覆盖天数超过90天的商品比例
- **滞销率**：超过90天无销售的库存比例  
- **售出率**：活跃FBA库存的销售效率
- **在售率**：所有FBA库存中有活跃listing的比例

**IPI阈值**：
- IPI ≥ 450：无限制，享受正常仓储容量
- IPI 400-449：可能收到容量限制警告
- IPI < 400：会被设置仓储容量上限，严重影响补货

**提高IPI分数的方法**：
1. 清理超过180天的库存（降价促销、移除销毁、亚马逊Outlet）
2. 降低过剩库存（减少补货量，让库存自然消化到合理水位）
3. 确保所有ASIN有活跃、可购买的Listing

### 2. 库存健康报告（Inventory Health Report）

每周查看关键指标：
- **库龄分布**：0-90天（正常）、91-180天（注意）、180天以上（危险）
- **月度仓储费估算**：提前预估费用
- **建议补货数量**：参考亚马逊的补货建议

## 补货公式与计划

### 核心补货计算

**补货触发点（Reorder Point）**：
```
补货触发库存量 = 日均销量 × (头程天数 + 安全库存天数)
```

**示例**：
- 日均销量：20单
- 海运头程：35天
- 安全库存：30天
- 补货触发点 = 20 × (35 + 30) = 1300件

当FBA可用库存降至1300件时，立即下采购单。

**补货数量（Order Quantity）**：
```
补货数量 = 目标覆盖天数 × 日均销量 - 当前库存
```

通常目标库存覆盖天数设为60-90天（避免超过90天产生长期仓储费）。

### 季节性备货调整

| 时期 | 备货倍数 | 备货截止时间 |
|------|---------|------------|
| 正常期 | 1.0x | 正常补货节奏 |
| Prime Day前 | 1.5-2.0x | Prime Day前45天 |
| 黑五/网一前 | 2.0-2.5x | 10月1日前 |
| 圣诞节前 | 1.5-2.0x | 11月15日前 |
| 节后清货期 | 减少补货 | 主动降价清库存 |

## 仓储费用管理

### 月度仓储费（Monthly Storage Fee）

按立方英尺计算：
- 标准尺寸（1-9月）：$0.75/立方英尺/月
- 标准尺寸（10-12月旺季）：$2.40/立方英尺/月
- 超大件（1-9月）：$0.48/立方英尺/月

**计算示例**（宠物睡垫，0.8立方英尺/件）：
- 100件库存 × 0.8 = 80立方英尺
- 旺季月仓储费 = 80 × $2.40 = $192/月

### 长期仓储费（Aged Inventory Surcharge）

- 181-270天库龄：$1.50/立方英尺
- 271-365天库龄：$6.90/立方英尺
- 365天以上：$6.90/立方英尺（或$0.15/件，取较高者）

**应对策略**：
- 181天前主动降价促销（降价20-30%）
- 参加"亚马逊再营销"项目
- 申请移除库存（支付移除费用，约$0.97/件）

2. **补货报告（Restock Report）**：亚马逊建议的补货量和时间
3. **库存健康报告**：详细的库龄分析

1. **Jungle Scout（库存管理器）**：自动计算补货点，发送补货提醒
2. **RestockPro**：专业库存管理和供应商PO管理
3. **卖家精灵库存预测**：基于历史销量的智能补货建议

## 断货预防策略

断货对自然排名的损害是持续性的，恢复排名需要1-4周时间，期间损失的销售额和排名价值是巨大的。

**多层次断货预防**：
1. **主要仓库（FBA）**：维持60天库存水位
2. **备用方案（FBM）**：当FBA库存不足10天时，激活FBM自发货选项
3. **紧急备货通道**：与供应商协议30天内可空运补货的紧急方案
4. **库存报警**：当可售天数低于45天时，发送邮件/短信提醒

## 实操建议

1. **建立库存管理日历**：每月1日检查IPI分数，每周一检查补货触发点
2. **与供应商沟通产能**：提前3-6个月告知大促备货需求，避免供应商产能不足
3. **分批次发货**：不要一次性将全部库存发往亚马逊，保留20%在本地仓库作为备用
4. **记录仓储历史数据**：每月记录库存成本，年终分析库存管理效率

## 总结

优秀的库存管理要做到"不断货、不积压、不过期"三个平衡。建立数据驱动的补货决策模型，结合季节性备货计划和断货预防机制，是实现FBA库存高效管理的核心方法。
